# Product

## Register

product

## Users

- Storage Provider operators checking node health, slot assignments, and earnings potential
- Crypto developers evaluating Shelby for AI data pipelines
- Web3 power users browsing network activity and blob storage
- All share one context: need data fast, no decoration, absolute clarity

## Product Purpose

ShelbyNet Explorer is the public monitoring layer for the Shelby decentralized hot storage protocol. It provides real-time visibility into storage provider health, network activity, and cost comparison. Without it, SP operators invest blind, developers can't assess the network, and the ecosystem lacks a trust signal.

## Brand Personality

**Precise. Trustworthy. Fast.**

Like Bloomberg Terminal or Grafana — numbers are the product. Every pixel serves legibility. Nothing decorative survives review.

## Anti-references

- DeFi purple/pink gradients
- Meme-coin aesthetics
- SaaS landing page templates (hero-text-cta-card grid)
- Glassmorphism
- Animated reveal sequences
- Rounded cards with shadow

## Design Principles

1. **Data first** — every pixel serves legibility and scan-ability
2. **Speed over decoration** — instant load, zero animation, no motion
3. **Professional trust** — nothing looks like a meme coin or marketing site
4. **Dense but scannable** — power users read tables and numbers, not cards and icons
5. **Honest tools** — show raw data, don't smooth it over

## Accessibility & Inclusion

- WCAG AA (minimum 4.5:1 contrast on body text)
- Keyboard navigation for all interactive elements
- Screen reader labels on data tables
- No motion (reduced-motion by default)
